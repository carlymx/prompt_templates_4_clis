# Qwen Code Assistant - Angular Project Setup

## Project Context

This project is designed to help users create Angular projects with customizable configurations through an interactive setup process. The configuration allows for various options including:

- Project name and structure
- Styling options (CSS/SCSS)
- Programming language choice (JavaScript/TypeScript)
- Ionic integration
- Database configuration (MongoDB, MariaDB, or local JSON)
- Target device platforms (Web, Android, iOS)
- Project templates (blank, minimal, basic with examples, or custom)

## Instructions for Qwen Code Assistant

When working with this Angular project:

1. **Language**: All responses and documentation should primarily be in Spanish (castellano) for the main README_ES.md file, with an English version in README.md.

2. **Project Setup**: The project uses Angular CLI and potentially Ionic Capacitor for multi-platform development.

3. **Database Options**: Support for MongoDB (NoSQL), MariaDB (relational), or local JSON file storage based on user requirements.

4. **Styling**: Support for both CSS and SCSS preprocessing.

5. **Programming Language**: Default to TypeScript for Angular projects with JavaScript as an alternative.

6. **Platform Support**: The project should be configurable for Web, Android, iOS, or all platforms using Capacitor.

7. **Project Templates**: Different starting templates are available from blank to basic examples with Ionic components.

## Security Notes

- Never include sensitive information like API keys, passwords, or personal data in project files
- Use environment variables for any sensitive configuration
- Follow Angular security best practices

## Development Guidelines

- Follow Angular style guide and best practices
- Use TypeScript by default for type safety
- Implement proper component structure and modularization
- Include proper testing for components and services
- Document code appropriately in Spanish and English
