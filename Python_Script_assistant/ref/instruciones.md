# Instrucciones para Qwen CLI

**Idioma:** Español (es-ES)

## Propósito del Proyecto

Este proyecto se enfoca en la creación de un Prompt Template para generar nuevos scripts Python con una estructura genérica común. La estructura extraída se encuentra en el archivo `estructura_genericas_scripts_python.md`.

## Componentes Principales

- `estructura_genericas_scripts_python.md`: Documento que describe la estructura genérica común a varios scripts Python de ejemplo.
- `README.md`: Documentación principal del proyecto (en inglés).
- `README_ES.md`: Documentación principal del proyecto (en español).
- `QWEN.md`: Instrucciones para iteraciones futuras (este archivo).
- `.chat_log/resumen_conversacion.md`: Resumen del proceso y conversaciones relevantes (a crear).

## Funcionalidades

- Extracción de patrones comunes de scripts Python.
- Documentación de buenas prácticas y estructuras típicas.
- Generación de plantillas de código reutilizables (Prompt Templates).

## Estructura del Proyecto

- `/` (raíz): Contiene los archivos de documentación principales y el archivo con la estructura genérica.
- `/.chat_log/`: Directorio para almacenar resúmenes de conversaciones (se creará).

## Instrucciones para Iteraciones Futuras

1. **Análisis de Nuevas Funcionalidades:** Si se agregan nuevos scripts de ejemplo, actualizar `estructura_genericas_scripts_python.md` con las nuevas características genéricas.
2. **Mantenimiento de Documentación:** Asegurar que `README.md` y `README_ES.md` reflejen cualquier cambio en la estructura o funcionalidades del proyecto.
3. **Registro de Cambios:** Documentar las conversaciones y decisiones importantes en `.chat_log/resumen_conversacion.md`.
4. **Consistencia de Enlaces:** Mantener los enlaces bilineales entre `README.md` y `README_ES.md` actualizados.
5. **Uso de Prompt Template:** Utilizar `estructura_genericas_scripts_python.md` como base para crear nuevos scripts Python con la estructura común identificada.
